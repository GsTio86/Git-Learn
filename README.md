# Git-Learn

* 初始化 Git
`git init`

* 加入修改或的紀錄
`git add <檔案>` 或 `git add .`

* 設定Commit
`git commit -m "Test Commit`

* 查看Git Log
`git log`

* 查看Git資料狀態
`git status`

* 下載遠端儲存庫
`git clone [ --recursive] <https url>`

* 連上遠端儲存庫
`git remote add origin <https url>`

* 選擇分支
`git branch -M <branch name>`

* 上傳到遠端儲存庫
`git push -u origin <branch name>`

* 更新遠端儲存庫到本機
`git pull`

* 下載遠端儲存庫新Commit
`git fetch`

* 查看遠端資訊
`git remote show origin`
